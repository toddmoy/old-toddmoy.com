# README

## Dependencies

* Node, for publishing to Surge
* Middleman, for generating the site
* Jquery, for SmoothState
* SmoothState, for nice page transitions
* Neat, for grids


## Working Server

```
middleman server
```


## Pushing to Surge.sh

Run the following

```
middleman build
cd build
surge
```
